Methodology

The mathematics behind shoulder-normalized jitter comparison.

01

Raw Displacement

Frame-to-frame Euclidean distance of each landmark in its native coordinate space.

d(f) = ‖p(f) − p(f−1)‖₂
02

Shoulder Width

Left-to-right shoulder distance in the same coordinate domain.

w(f) = ‖shoulder_R − shoulder_L‖
03

Normalization

Divide displacement by median shoulder width — a dimensionless, body-relative metric.

J(f) = d(f) / median(w)
04

Cross-Domain

Compare MediaPipe (0–1 normalized) and stereo vision (mm) using the same dimensionless unit.

No false unit conversion needed
"Jitter was computed as the Euclidean displacement of each landmark between consecutive frames within its native coordinate domain. Each displacement was divided by shoulder width measured in the same domain, yielding a dimensionless, body-relative metric suitable for cross-system comparison."
